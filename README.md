Description:
What it is:
An ready to use software capable of detecting peers connected to an magnet/torrent swarm, the software comes with additional features for searching for magnets from multiple sites at once, and tracking peers by hosting your own file.

How it works:
For the peer detection & hosting and tracking, it uses qBittorrent and ipwhois, to find out location and related info of the connected peers. For Searching for magnets, it uses Jakett.

Instructions for use:
1. Extract the contents of MSMT.zip
2. copy the MTSM folder from assets into the path "C:\Users\GANESH\AppData\Roaming\"
3. configure Jackket and qBittorrent ports as set in the settings.
enjoy!
